# rdb
### Revision database for A-levels.
![VKSL Logo|500x500](content/assets/logo.svg)

A collaborative effort to compile revision resources for
- A-level Computer Science (AQA 7517)
- A-level Maths (OCR A H240)
- A-level Further Maths (OCR A H245)
- A-level Physics (OCR B Advancing Physics H557)

### Contributing
Any contribution to this repo would be greatly appreciated, including fixes for typos or other small changes. The recommended workflow is using a tool such as Obsidian to edit the Markdown files.
