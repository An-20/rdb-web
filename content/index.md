#### About RDB Web

RDB Web is a copy of [my](https://github.com/An-20) A-level notes. The notes are available in source Markdown format under the MIT license at the [GitHub repository](https://github.com/An-20/rdb-web). The notes are converted from Markdown to HTML using the excellent [Obsidian Webpage Export plugin](https://github.com/KosmosisDire/obsidian-webpage-export).
